# 单骑护主 PWA

一款可安装、可离线游玩的原创汉字合成塔防网页 App。游戏采用固定比例的 7×4 阵位和三层蛇形战线，包含弓、枪、术、鼓、弩、刀六种兵牌，以及普通敌军、骑兵、盾兵、魁兵、旗兵、医兵和敌将。

六个兵种具有独立攻击表现：弓兵发射弓箭、枪兵近程突刺、术士投射火焰并产生爆炸、鼓兵释放声波、弩兵发射破甲重弩、刀兵近身横扫并收割低血量敌军。

## 发布候选版功能

- 三级难度：演义、鏖战、无双。
- 四阶武将晋升：黄忠、赵云、诸葛亮、刘备、姜维、关羽。
- 六项武将专属被动和独立金色兵牌效果。
- 旗兵加速军阵、医兵治疗、敌将周期激励等特殊敌军机制。
- 每三波一次兵法三选一，支持多种局内构筑。
- 军械库局外成长：战后获得虎符，永久强化开局资源、安危上限、伤害、征兵费用、技能冷却和虎符收益。
- 战场军令：每局随机变体，改变敌军强度、速度、数量、军粮收益和军功倍率。
- 战场悬赏：每局随机两个目标，完成后结算额外虎符。
- 连斩、结阵、战鼓光环、主动技能和 20 波完整流程。
- 主动技能包含龙胆破阵、伏兵截道和军粮急送，支持键盘 `1`、`2`、`3` 触发。
- 兵将图鉴、暂停菜单、难度记录、最高分、通关次数和武将解锁记录。
- 原创长坂坡水墨战场背景、强化兵牌武器装饰和敌军兵种视觉识别。
- 战绩分享、安装提示、离线缓存、自动失焦暂停和移动端横屏提示。
- iPhone Safari 主屏幕版：支持安全区适配、横屏紧凑布局、触控拖拽优化和离线启动。

## 本地预览

请勿直接双击 `index.html` 测试安装功能，因为浏览器只允许网页 App 在 HTTPS 或 localhost 下注册离线服务。

在本目录打开终端并运行：

```bash
python3 -m http.server 8080
```

随后访问 `http://localhost:8080`。

战斗逻辑回归测试：

```bash
node tests/combat-smoke.mjs
```

生成静态发布目录：

```bash
npm run build
```

## 部署

将整个文件夹原样上传到支持 HTTPS 的静态网站服务即可，例如 GitHub Pages、Cloudflare Pages、Netlify 或 Vercel。不要只上传 `index.html`。

## 安装

- Android / Chrome：打开网址，点击游戏右上角“安装 App”，或在浏览器菜单选择“安装应用”。
- iPhone / Safari：打开网址，点击游戏右上角“添加到主屏幕”查看步骤；也可以直接点 Safari 分享按钮，选择“添加到主屏幕”。
- 电脑 / Chrome 或 Edge：打开网址，点击地址栏右侧的安装图标。

首次联网打开后，游戏资源会被缓存，之后可以离线启动。

## iPhone 发布方式

当前交付形态是 PWA，不需要经过 App Store。把整个目录部署到 HTTPS 静态站后，玩家用 iPhone Safari 打开网址并添加到主屏幕，即可像 App 一样全屏启动。

如果后续要上架 App Store，需要再做一个 iOS 原生壳：用 Xcode 创建 iOS App，内置 WKWebView，加载这套静态资源，并配置签名、隐私说明、年龄分级和截图。
