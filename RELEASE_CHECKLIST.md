# 发布检查清单

## 游戏

- [ ] 三种难度各完成至少 5 局真机测试。
- [ ] 六个普通兵种和六名武将均完成攻击、合成和被动检查。
- [ ] 旗兵、医兵、盾兵、魁兵和四次 Boss 波均完成检查。
- [ ] 验证断网重开、后台恢复、暂停、重新开始和战绩保存。
- [ ] 记录各难度通关率、平均结束波次和平均局长，再做最终平衡。

## 设备与浏览器

- [ ] iPhone Safari：竖屏提示、横屏游玩、添加到主屏幕和离线启动。
- [ ] Android Chrome：安装、触控拖拽、返回键和离线启动。
- [ ] macOS Safari/Chrome 与 Windows Edge/Chrome。
- [ ] 检查低分辨率手机、平板和高 DPI 屏幕。

## 发布

- [ ] 使用 HTTPS 托管整个文件夹。
- [ ] 确认 `manifest.webmanifest`、`sw.js` 和全部图标返回 200。
- [ ] 确认隐私说明与实际使用的分析、广告或账户功能一致。
- [ ] 准备商店或网站截图、游戏简介、年龄分级和支持联系方式。
- [ ] 发布后检查 Service Worker 更新是否能从旧版本平稳升级。
